# 🏠 FlatMate Expense Manager

A modern and intuitive web application to manage shared expenses among flatmates. Track daily spending, split costs fairly, and get real-time insights into who owes what — all in one place.

## 🚀 Features

- ➕ Add, edit, and delete expenses
- 👥 Manage multiple flat members
- 📊 Monthly summaries and analytics
- 📈 Interactive charts (category-wise & member-wise)
- 🔍 Advanced filtering, sorting, and search
- 💰 Automatic expense splitting and balance calculation
- 🧾 Rent and mess contribution tracking
- 📱 Fully responsive design (mobile-friendly)
- ⚡ Real-time updates (with backend integration)

## 🛠️ Tech Stack

- **Frontend:** React (via CDN)
- **Charts:** Chart.js
- **Styling:** Custom CSS
- **Storage:**
  - LocalStorage (default)
  - Firebase Firestore (for real-time multi-user support)

## ⚙️ How It Works

- Expenses are tracked per member with categories
- Monthly rent and mess contributions are configurable
- The app calculates:
  - Total expenses
  - Individual contributions
  - Balances (who owes / who gets refunded)
- Data can be stored locally or synced across users using Firebase
